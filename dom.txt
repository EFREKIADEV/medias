DOCUMENT OBJECT MODEL (DOM)

É uma interface que representa documentos HTML e XML através
de objetos. Com ela é possível manipular a estrutura, estilo
e conteúdo destes documentos.


WINDOW 
    - LOCATION
    - DOCUMENT
        - HTML
            - HEAD
            - BODY
                - H1
                - p
                - img
    - HISTORY



SELETORES NO JAVASCRIPT
 - ID                       document.getElementByID
 - CLASSE E TAG             document.getElementsByClassName getElementsByTagName
 - SELETOR GERAL ÚNICO      document.querySelector
 - SELETOR GERAL LISTA      document.querySelectorAll
